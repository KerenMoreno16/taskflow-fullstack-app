# TaskFlow

Modern full stack task management application built with React, Node.js, Express, and MongoDB Atlas.

---

## Features

- Create tasks
- Edit tasks
- Delete tasks
- REST API
- MongoDB Atlas integration
- Responsive UI
- Modern dashboard design

---

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Axios
- Lucide React

### Backend
- Node.js
- Express
- MongoDB Atlas
- Mongoose

### Frontend

